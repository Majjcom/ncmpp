# ncmpp

一个极速的多线程ncm解密器

建议使用带有图形界面的版本：[ncmppGui](https://github.com/Majjcom/ncmppGui)

## 使用方法

### 快速开始

- 下载`Release`中最新版本的可执行文件

- 将文件复制到有`ncm`文件的目录中

- 双击运行程序

- 等待解密完成
- 解密后的文件存放在`unlock`文件夹中

### 命令行参数

--threads[t] 最大线解密程数

--showtime[s] 显示解密消耗的时间

## 构建环境

Windows 10

Visual Studio 2022

SDK: 10.0.19041.0

openssl: 1.1.1s
